# 허쉬 포트폴리오

**3D Creative • Seoul**

## 컨셉

감도 높은 아티스트 웹사이트. 순수한 미니멀리즘과 타이포그래피 중심의 갤러리 경험.

## 특징

- **완전 미니멀** - 블랙 배경, 화이트 텍스트
- **타이포그래피 중심** - 프로젝트 이름이 주인공
- **인터랙티브 호버** - 마우스 오버시 프로젝트 상세 정보
- **커스텀 커서** - 브랜드 경험 강화
- **갤러리 감성** - 아티스트다운 포트폴리오

## 기술

- 바닐라 HTML/CSS/JavaScript
- Inter 폰트
- CSS 그라디언트 & 애니메이션
- 반응형 디자인
